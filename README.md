# SauceDemo Automation Testing (Python + Selenium)

This script automates the basic user flow of the SauceDemo website using Selenium WebDriver in Python.

## Functionalities Automated
- Login with valid credentials
- View product listings
- Add item to cart
- Open cart page and verify item
- Logout workflow

## Tools & Technologies
- Python
- Selenium WebDriver
- ChromeDriver

This project demonstrates beginner-level automation skills and understanding of Selenium locators, browser control, and basic validations.
